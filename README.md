# Seoul5-DigitalTwin-Population
Synthetic population dataset and code for Seoul 5-districts digital twin (2024)

This repository contains the R code and synthetic population datasets
for constructing a 3.7-million-record digital twin population for five districts 
in Seoul, Republic of Korea (Geumcheon, Guro, Gwanak, Yeongdeungpo, Dongjak).

## Contents
- data/: synthetic datasets
- code/: full reproducible R script
- docs/: data dictionary
